# Pitwall — Project Overview

## What this project is
An end-to-end F1 data engineering and machine learning project built on PySpark and Databricks. It ingests Formula 1 session data, cleans and enriches it, engineers predictive features, and trains a model to forecast race winners. A public dashboard displays predictions with confidence scores.

## Core goal
Predict F1 race winners using practice, qualifying, sprint, and historical race data — with a model that favours recent performance to account for mid-season upgrades and car development.
